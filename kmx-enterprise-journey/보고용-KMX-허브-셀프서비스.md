# 0422 KMX 기업파트 정리

브라우저 **입력 폼·파일 업로드**로 기업 가입·데이터 등록 등을 하면, **같은 서버 안의 API**가 처리하고 **JSON 파일 + 업로드 폴더**에 남깁니다.

---

## 1. 구성


| 항목         | 내용                                                              |
| ---------- | --------------------------------------------------------------- |
| **시작 페이지** | `http://(서버):8090/app/index.html` — 루트 `/` 는 여기로 자동 이동          |
| **구성**     | **왼쪽:** 기업 담당자(등록·데이터·파일·DB 프로필 등) / **오른쪽:** 허브(승인·감사) — 역할 분리 |
| **기술**     | 일반 웹 페이지(HTML/CSS/JS) + 서버 API 호출. 별도 프론트 빌드 없음                 |


---

## 2. Back

```
[브라우저]
    │  (1) 페이지 열기: 정적 HTML
    ▼
/app/*.html  ──서버에 파일로 있음──►  hub/static/app/

[브라우저]
    │  (2) 제출·목록: JSON 또는 파일(multipart)
    ▼
/api/...       ──FastAPI 라우트──►  hub/main.py
                      │
                      ▼
              hub/logic.py (규칙: pending만 승인 등)
                      │
                      ▼
              hub/store.py → data/hub_state.json
              업로드 파일 → data/uploads/{기업번호}/
```

- **같은 프로세스**에서 정적 파일 제공과 API가 같이 동작합니다 (`uvicorn hub.main:app`).
- **로그인·SSO는 없음**(MVP). 누가 했는지는 일부 API에서 `X-Actor-Email` 헤더로 감사에만 기록.

---

## 3. 프로젝트 세부 파트


| 위치                        | 역할                                                         |
| ------------------------- | ---------------------------------------------------------- |
| `**hub/static/app/`**     | 사용자가 보는 **HTML·CSS·JS** (`index.html`, `register.html`, …) |
| `**hub/main.py`**         | **URL 연결**: `/` 리다이렉트, `/api/*` API, `/app` 정적 마운트         |
| `**hub/logic.py`**        | **비즈니스 규칙** (예: 승인된 기업만 파일 업로드)                            |
| `**hub/store.py`**        | `**data/hub_state.json**` 읽기/쓰기, 감사 이벤트 추가                 |
| `**hub/schemas.py**`      | API 요청/응답 **형식 검증** (Pydantic)                             |
| `**hub/data_files.py`**   | 업로드 **디스크 경로** (`data/uploads/`)                           |
| `**run_hub.py`**          | 서버 실행 편의 스크립트 (`0.0.0.0:8090`)                             |
| `**data/hub_state.json**` | 실행 후 생기는 **저장 데이터**(git 제외 권장)                             |
| `**data/uploads/`**       | 업로드된 **파일 본문**                                             |


**이 폴더 밖:** `catena-x` 등은 이번 셀프서비스와 **직접 연결 안 함**. 다음 단계에서 API 훅으로 붙일 수 있음.
